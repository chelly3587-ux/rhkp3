1(연습문제 9.1-2) 평면의 방정식
* 1단계 (What): 두 점 $P_1(a_1, b_1, c_1)$과 $P_2(a_2, b_2, c_2)$ 사이의 수직이등분면을 구하는 문제이다. 점 $P(x, y, z)$ 가 두 점으로부터 같은 거리에 있어야 하므로 거리 공식 $\sqrt{(x-a_1)^2+(y-b_1)^2+(z-c_1)^2} = \sqrt{(x-a_2)^2+(y-b_2)^2+(z-c_2)^2}$ 를 세운다. 양변을 제곱하여 정리하면 $2(a_2-a_1)x + 2(b_2-b_1)y + 2(c_2-c_1)z = (a_2^2+b_2^2+c_2^2) - (a_1^2+b_1^2+c_1^2)$라는 선형 방정식을 얻는다.근거: 점과 점 사이의 거리 공식, 평면의 정의.
* 2단계 (So What): 이 방정식은 두 점을 잇는 선분을 수직이등분하는 평면을 의미한다.
