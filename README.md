1(연습문제 9.1-2) 평면의 방정식
* 1단계 (What): 두 점 $P_1(a_1, b_1, c_1)$과 $P_2(a_2, b_2, c_2)$ 사이의 수직이등분면을 구하는 문제이다. 점 $P(x, y, z)$ 가 두 점으로부터 같은 거리에 있어야 하므로 거리 공식 $\sqrt{(x-a_1)^2+(y-b_1)^2+(z-c_1)^2} = \sqrt{(x-a_2)^2+(y-b_2)^2+(z-c_2)^2}$ 를 세운다. 양변을 제곱하여 정리하면 $2(a_2-a_1)x + 2(b_2-b_1)y + 2(c_2-c_1)z = (a_2^2+b_2^2+c_2^2) - (a_1^2+b_1^2+c_1^2)$라는 선형 방정식을 얻는다. 점과 점 사이의 거리 공식, 평면의 정의.
* 2단계 (So What): 이 방정식은 두 점을 잇는 선분을 수직이등분하는 평면을 의미한다.


2 (연습문제 9.1-3) 아폴로니우스 구
* 1단계 (What): 두 점 $U, V$ 로부터의 거리 비가 $m:n$인 점 $P$ 의 자취를 구한다. $\frac{|\vec{UP}|}{|\vec{VP}|} = \frac{m}{n}$ 에서 $n^2|\vec{UP}|^2 = m^2|\vec{VP}|^2$ 를 전개한다. 이를 정리하면 $(x-h)^2 + (y-k)^2 + (z-l)^2 = r^2$ 형태의 구의 방정식이 도출된다.
* 2단계 (So What): 2차원에서의 '아폴로니우스의 원'이 3차원 공간으로 확장되어 '구'를 형성함을 보여준다.


3 (연습문제 9.1-8) 벡터의 선형 결합
*1단계 (What): $v = c_1u_1 + c_2u_2 + c_3u_3$ 를 만족하는 상수를 찾기 위해 연립방정식을 세운다. $\begin{cases} c_1 + c_3 = 1 \\ 2c_2 + 2c_3 = 1 \\ 3c_1 + c_2 = 1 \end{cases}$ 이를 풀면 $c_1 = \frac{1}{8}, c_2 = \frac{5}{8}, c_3 = \frac{7}{8}$ 이므로 $v = \frac{1}{8}u_1 + \frac{5}{8}u_2 + \frac{7}{8}u_3$ 이다.근거: 벡터의 선형 결합 정의, 가우스 소거법.
2단계 (So What): 임의의 벡터 $v$ 를 기저 벡터들의 합으로 분해하는 과정이다.
